# Blood Donation Management System

## Project Overview

The Blood Donation Management System is a web-based application developed using the Django framework. The system is designed to simplify the management of blood donors, blood donations, blood requests, and blood inventory. It provides an efficient way for administrators to maintain donor records and manage blood availability.

## Objectives

* Register and manage blood donors.
* Record blood donation information.
* Manage blood requests from patients or hospitals.
* Maintain blood inventory records.
* Provide an administrator interface for managing all system data.

## Features

* Donor registration
* Donor information management
* Blood donation tracking
* Blood request management
* Blood group management
* Django administration panel
* SQLite database integration

## Technologies Used

* Python 3
* Django
* SQLite
* HTML
* CSS
