Student Grade Calculator Web Application
Project Title

Student Grade Calculator Using DOM Manipulation

Objective

The objective of this project is to develop a dynamic web application that captures student marks, processes them using JavaScript, and instantly displays the corresponding grade using DOM manipulation without refreshing the page. This project demonstrates the use of HTML, CSS, and JavaScript in building interactive web applications.

Purpose

The purpose of this application is to assist lecturers and students in quickly calculating academic grades from marks in real time. It reduces manual grading effort, improves accuracy, and provides instant feedback to users.

Tools Used

HTML5 – for structuring the webpage
CSS3 – for styling and layout design
JavaScript – for logic, validation, and DOM manipulation
Visual Studio Code – code editor
Google Chrome – testing browser

Challenges Faced
Handling invalid and empty inputs
Implementing correct conditional logic
Selecting DOM elements correctly using IDs
Updating results dynamically without page reload
Resetting input field after each calculation
Conclusion

This project demonstrates how JavaScript DOM manipulation can be used to build interactive web applications. It strengthens understanding of event handling, input validation, and real-time data processing in web development.
